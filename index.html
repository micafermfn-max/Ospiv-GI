URL del Apps Script (GAS)
Cada vez que redespleguës el script en Google Apps Script obtenés una URL nueva. Pegala acá y guardá — el sistema la va a usar a partir de ese momento sin necesidad de tocar el código.
Cómo obtener la URL
  1. Abrí el proyecto en script.google.com
  2. Hacé click en Implementar → Nueva implementación
  3. Tipo: Aplicación web · Acceso: Cualquier persona
  4. Copiá la URL que aparece y pegala arriba
